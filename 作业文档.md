# 博客管理系统

## 项目简介

基于 Vue + Element UI 的博客后台管理系统，用于管理文章、分类、标签、友链、评论等内容。

## 技术栈

- Vue 2.6
- Element UI 2.15
- Vuex 3.6
- Vue Router 3.5
- Axios

## 项目结构

```
src/
├── api/          # 接口请求
│   ├── article.js    # 文章管理
│   ├── category.js   # 分类管理
│   ├── comment.js    # 评论管理
│   ├── link.js       # 友链管理
│   ├── login.js      # 登录认证
│   ├── menu.js       # 菜单管理
│   ├── role.js       # 角色管理
│   ├── user.js       # 用户管理
│   └── tag.js        # 标签管理
├── views/        # 页面视图
│   ├── article/      # 文章管理页面
│   ├── category/     # 分类管理页面
│   ├── comment/      # 评论管理页面
│   ├── link/         # 友链管理页面
│   ├── menu/         # 菜单管理页面
│   ├── role/         # 角色管理页面
│   ├── user/         # 用户管理页面
│   ├── layout/       # 布局组件
│   └── login/        # 登录页面
├── layout/       # 系统布局
├── router/       # 路由配置
├── store/        # 状态管理
└── utils/        # 工具函数
```

## 后端服务

| 服务 | 端口 | 说明 |
|------|------|------|
| admin-server | 8989 | 用户认证、菜单、角色等系统接口 |
| blog-server | 7777 | 文章、分类、标签、友链、评论等内容接口 |

## 启动命令

```bash
# 安装依赖
npm install

# 启动开发服务器 (端口 9528)
npm run dev

# 构建生产版本
npm run build:prod

# 测试 API 接口
node test.js
```

## 功能模块

1. 文章管理 - 文章列表查看
2. 分类管理 - 分类列表查看
3. 评论管理 - 评论列表查看
4. 友链管理 - 友链列表查看
5. 菜单管理 - 菜单树形展示
6. 角色管理 - 角色列表查看
7. 用户管理 - 用户列表展示
